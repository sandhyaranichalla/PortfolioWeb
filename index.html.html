HerSafeNest

Hostel Management System · MERN Stack

Problem Solved Hostels relied on scattered spreadsheets and paper records to manage students, rooms, fees, and complaints — leading to errors, inefficiency, and no visibility for residents.

A centralised multi-role hostel management platform that replaces manual operations with a single digital system. Admin and Student roles each access a customised dashboard with restricted, role-appropriate data — enforced at the API level.

Key Features

  • JWT + RBAC middleware: Admin/Student roles with zero unauthorized data exposure
  • 10+ REST API endpoints with structured JSON, HTTP status codes, modular routing
  • Real-time room allocation engine preventing concurrent double-booking server-side
  • Automated fee status classification (Paid / Unpaid) from payment records
  • Complaint lifecycle management: Pending → In Progress → Resolved
  • MongoDB schemas with cross-collection document references and referential integrity
  • MVC-structured backend: routing, controllers, and Mongoose models cleanly separated

E-Commerce Web Application · MERN Stack

Problem Solved Building a reliable e-commerce platform requires seamless coordination between a dynamic frontend and a secure, scalable backend — with real-time state and protected transactions.

A production-grade e-commerce platform with end-to-end functionality: registration, dynamic product catalogue, cart management, and order processing. React frontend integrated with a Node.js/Express.js backend via async REST API calls.
